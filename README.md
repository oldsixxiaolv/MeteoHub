# MeteoHub 🚀

**MeteoHub** 是专为大气科学领域学生和研究者打造的学术交流平台。

> 由吕亦航创建于中国科学院大气物理研究所

---

## 🚀 快速启动（任选一种方式）

### 方式 1：后台运行（最简单，推荐）

```bash
cd MeteoHub

# 一键后台启动
./run-background.sh

# 然后访问：
# http://120.46.134.210:8080
```

服务会在后台持续运行，关闭终端也不会停止。

### 方式 2：系统服务（开机自动启动）

```bash
cd MeteoHub
sudo ./install-service.sh
```

这种方式会在开机时自动启动服务，最稳定。

### 方式 3：前台运行（调试用）

```bash
./start.sh
```

---

## 📌 管理命令

```bash
# 查看服务状态
./status.sh

# 停止服务
./stop.sh

# 重启服务
./restart.sh

# 查看日志
tail -f server.log
```
